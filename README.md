# werwolf-online
A simple online werwolves game

(currently in developement)

## Installation

### Requirements
```
Node.js>=v15.0.9
npm>=7.5.2
```

### Installing
```bash
git clone https://github.com/cephox/werwolf-online
cd werwolf-online
npm i
(npm i -g typescript)
tsc
```

## Running
```bash
npm start
```
